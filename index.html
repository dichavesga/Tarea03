Información General

El diseño web responsivo permite adaptar una página a cualquier dispositivo.

Tecnologías como Viewport, Media Queries y Bootstrap facilitan la creación de interfaces modernas.

Diseño Web Responsivo

Es una metodología de diseño que permite adaptar un sitio web a diferentes tamaños de pantalla.

Su objetivo es mejorar la experiencia de usuario en computadoras, tabletas y teléfonos móviles.

  • Mejor experiencia de usuario.
  • Mayor accesibilidad.
  • Optimización SEO.
  • Menor mantenimiento.

Viewport

El viewport corresponde al área visible del navegador.

Es fundamental para controlar la visualización en dispositivos móviles. 

<meta name="viewport"
content="width=device-width, initial-scale=1.0">

CSS3 Media Queries

Permiten aplicar estilos distintos según el tamaño de la pantalla. 

@media (max-width:768px){
    .container{
        flex-direction:column;
    }
}

Bootstrap 5

Framework CSS que facilita la construcción de sitios web responsivos mediante componentes prediseñados.

  • Containers
  • Grid System
  • Flex Utilities
  • Carousel
  • Navbar

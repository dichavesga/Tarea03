Información General

El diseño web moderno requiere que los sitios funcionen correctamente en computadoras, tabletas y teléfonos móviles.

Gracias a tecnologías como Viewport, Media Queries y Bootstrap es posible construir interfaces adaptables.

Diseño Web Responsivo

El diseño web responsivo es una técnica que permite adaptar automáticamente una página web a distintos tamaños de pantalla.

Su objetivo es ofrecer una experiencia de usuario óptima independientemente del dispositivo utilizado.

Importancia

  • Mejora la experiencia del usuario.
  • Facilita la navegación.
  • Favorece el posicionamiento SEO.
  • Reduce costos de mantenimiento.

Viewport

El viewport representa el área visible que el usuario observa dentro del navegador.

Su configuración permite controlar cómo se muestra el contenido en dispositivos móviles. 

<meta name="viewport"
content="width=device-width, initial-scale=1.0">

CSS3 Media Queries

Las Media Queries permiten aplicar estilos diferentes según el tamaño de la pantalla.

Son una de las herramientas más importantes para crear sitios web responsivos. 

@media (max-width:768px){
    .container{
        flex-direction:column;
    }
}

Bootstrap 5

Bootstrap es un framework que facilita el desarrollo de sitios web modernos y adaptables.

Componentes principales
